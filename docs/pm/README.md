# Cardra PM 문서 허브

## 문서 목록
- `cardra-pm-overview.md` : 진행요약 및 PM 우선순위
- `cardra-agent-architecture-spec.md` : Agent Orchestrator + 스키마/에러·fallback 정책
- `cardra-researcher-pipeline.md` : 리서처 파이프라인 및 API 계약
- `cardra-content-orchestrator.md` : 카드 생성 규칙(1~3장), 톤/검수 기준
- `cardra-designer-guidelines.md` : Nano Banana 기준 레이아웃/톤 가이드
- `cardra-recommender-mvp.md` : 추천 엔진 MVP 설계

## 사용법
- 코드 이슈/백로그를 생성할 때는 해당 문서를 링크로 첨부 후, "영향 파일"과 "테스트 변경"을 함께 적기
- PM 승인 시 문서 기준대로 우선순위(안전성 > 품질 > 확장성)로 진행
