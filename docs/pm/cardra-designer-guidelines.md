# Cardra 카드뉴스 디자이너 가이드 (Nano Banana 톤)

## 목표
프론트/디자인 팀이 바로 사용 가능한 카드뉴스 템플릿 규칙 정리.

## 톤 & 스타일
- 경쾌·미니멀·클린·가독성 최우선
- 폰트: Pretendard / SUIT / Noto Sans KR
- 비율: 인스타 1080×1350(4:5), 범용 1080×1080
- 색: 밝은 배경 + 포인트 1색(옐로우/민트/코발트)
- 아이콘/일러스트 스타일 통일

## 카드별 템플릿
### 1장 (Hook)
- 제목: 10~18자, 본문: 0~30자(선택)
- 라벨: `TREND` / `AI TIP` / `3분요약`
- 텍스트:이미지 ≈ **55:45**

### 2장 (핵심)
- 제목: 12~22자, 본문: 70~130자
- 라벨: `HOW TO` / `핵심포인트`
- 텍스트:이미지 ≈ **65:35**

### 3장 (CTA)
- 제목: 10~16자, 본문: 30~70자
- 라벨: `CHECK` / `요약` / `CTA`
- 텍스트:이미지 ≈ **50:50**

## 공통 규칙
- 최소 여백 64px
- 제목/본문/라벨의 계층 구조 고정
- 카드당 핵심 메시지 1개
- 출처 라벨이나 근거 라벨은 텍스트 대비 높게

## 바로 적용 예시
- 표지: `요즘 AI, 이렇게 써야 이득` / `3분만에 핵심만 정리`
- 설명: `실수 줄이는 프롬프트 3원칙`
- 마무리: `오늘부터 바로 적용해보세요` + CTA 버튼

## 코드/프론트 팀 전달 포인트
- backend는 카드 텍스트 + source 배열/라벨만 안정적으로 보내면 됨
- 시각적 연동 필드 권장:
  - `variant`: 템플릿 타입 (`headline | insight | summary`)
  - `style`: `tone/layout/emphasis`
  - `media`: `imageUrl/imageType/altText`
  - `cta`: `label/actionType/target`
  - `tags`: 키워드/룩앤필 태그
  - `imageHint`: 이미지 생성 힌트
- 현재 MVP에서는 backend에서 상기 필드를 전달 시작(이미지 생성 모델 연동은 다음 단계)
